Project Title: Parallel Techniques for Biological Sequence Alignment Tools

Project Outline:

Sequence alignment involves comparing multiple strings of letters (representing nucleotides or amino acids) and computing the similarity between them. A very common tool to perform this operation is the Basic Local Alignment Search Tool (BLAST). Using BLAST, one can submit a query sequence to be compared with entire databases of sequences and return similarity scores. With next-generation sequencing fueling growing databases and often multiple queries being submitted, parallel approaches have become a necessity to reduce processing time.

Firstly, an assessment of parallel approaches to speeding up sequence alignment tools (BLAST) will provide an overview of current solutions and novelties of different approaches. These may include hardware acceleration with GPUs and FPGAs, data distribution methods using MPI, or other novel methods. Secondly, the most recent, best performing, and available techniques for re-implementation will be considered for comparison on various query and database configurations. Finally, the performance of applied approaches will be evaluated and may lead to configuration-specific recommendations or hybrid solutions for parallelization.
